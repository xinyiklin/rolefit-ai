# RoleFit Local Companion 0.1.0 — unsigned preview beta.1

<!-- Curated preview-note source. Publication occurs only after every native build, packaged smoke, installer lifecycle, checksum, and atomic-publication gate succeeds. -->

This is an unsigned public preview of the first RoleFit AI desktop companion.
It is intended for early testing while project-owned Apple and Windows signing
identities are unavailable.

## Before installing

- macOS Gatekeeper will identify the app as coming from an unidentified
  developer. The macOS app has a local ad-hoc integrity signature, but it is
  not notarized or signed with Apple Developer ID.
- Windows SmartScreen will warn that the publisher is unknown. The installer
  is not Authenticode-signed.
- Download only from the canonical RoleFit GitHub Release and compare the file
  against `SHA256SUMS.txt` before opening it.

## Included

- Native macOS downloads for Apple silicon and Intel.
- A native Windows x64 installer.
- Local setup for Codex, Claude Code, and Antigravity CLI providers.
- Operating-system-encrypted storage for configured OpenAI and Anthropic API
  keys; saved keys are never returned to the browser UI.
- A configurable local RoleFit site port, defaulting to `5181`.
- The RoleFit Drafting Desk in the normal browser, served only from the
  companion-owned loopback server.

## Data boundary

RoleFit has no app account, hosted workspace, cloud synchronization, or
telemetry service. Workspace resumes, jobs, tracker records, and generated
artifacts stay in the local RoleFit workspace. When you run an AI action, the
inputs required for that action are sent directly to the CLI or API provider
you selected.

## Browser extension

The unpacked browser extension requests local access on first use. Approve its
exact browser/profile origin once in the companion; unapproved extensions
cannot analyze or import a posting, and removing the pairing revokes access.

Every downloadable asset is produced by its native platform preview job. The
release includes `SHA256SUMS.txt` for independent checksum verification.
